Our plugin can be easily installed and used by anyone, from the technical novice to the technical master. Installation is as easy as cut and paste, and the form is completely customizable. == Installation ==


Using The WordPress Dashboard

* Navigate to "Plugins" -> Click "Add New" button
* Search for 'Telemed Payments'
* Click "Install Now"
* Click the "Activate Plugin" link

Uploading via WordPress Dashboard

* Navigate to "Plugins" -> Click "Add New" button
* Click the "Upload Plugin" button
* Browse to the telemed-payments.zip download location on your computer
* Click "Install Now"
* Click the "Activate Plugin" link

Contact Telemed Processing at https://telemedprocessing/contact-us/ to request sandbox api key for testing and production key for live sales.
